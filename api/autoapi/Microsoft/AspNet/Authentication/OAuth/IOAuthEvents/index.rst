

IOAuthEvents Interface
======================



.. contents:: 
   :local:



Summary
-------

Specifies callback methods which the OAuthMiddleware invokes to enable developer control over the authentication process.











Syntax
------

.. code-block:: csharp

   public interface IOAuthEvents : IRemoteAuthenticationEvents
