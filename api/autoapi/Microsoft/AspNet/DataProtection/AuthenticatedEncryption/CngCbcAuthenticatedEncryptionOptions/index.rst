

CngCbcAuthenticatedEncryptionOptions Class
==========================================



.. contents:: 
   :local:



Summary
-------

Options for configuring an authenticated encryption mechanism which uses
Windows CNG algorithms in CBC encryption + HMAC authentication modes.
