

CngCbcAuthenticatedEncryptorConfiguration Class
===============================================



.. contents:: 
   :local:



Summary
-------

Represents a configured authenticated encryption mechanism which uses
Windows CNG algorithms in CBC encryption + HMAC authentication modes.
