

ManagedAuthenticatedEncryptionOptions Class
===========================================



.. contents:: 
   :local:



Summary
-------

Options for configuring an authenticated encryption mechanism which uses
managed SymmetricAlgorithm and KeyedHashAlgorithm implementations.
