

HostString Struct
=================



.. contents:: 
   :local:



Summary
-------

Represents the host portion of a URI can be used to construct URI's properly formatted and encoded for use in
HTTP headers.











Syntax
------

.. code-block:: csharp

   public struct HostString : IEquatable<HostString>
