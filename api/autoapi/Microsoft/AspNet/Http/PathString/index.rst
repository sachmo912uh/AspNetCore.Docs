

PathString Struct
=================



.. contents:: 
   :local:



Summary
-------

Provides correct escaping for Path and PathBase values when needed to reconstruct a request or redirect URI string











Syntax
------

.. code-block:: csharp

   public struct PathString : IEquatable<PathString>
