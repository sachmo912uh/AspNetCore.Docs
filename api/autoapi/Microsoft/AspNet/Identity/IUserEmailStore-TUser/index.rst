

IUserEmailStore<TUser> Interface
================================



.. contents:: 
   :local:



Summary
-------

Provides an abstraction for the storage and management of user email addresses.











Syntax
------

.. code-block:: csharp

   public interface IUserEmailStore<TUser> : IUserStore<TUser>, IDisposable where TUser : class
