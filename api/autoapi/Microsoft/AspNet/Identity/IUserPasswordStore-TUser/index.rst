

IUserPasswordStore<TUser> Interface
===================================



.. contents:: 
   :local:



Summary
-------

Provides an abstraction for a store containing users' password hashes..











Syntax
------

.. code-block:: csharp

   public interface IUserPasswordStore<TUser> : IUserStore<TUser>, IDisposable where TUser : class
