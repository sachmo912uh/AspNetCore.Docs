

IUserStore<TUser> Interface
===========================



.. contents:: 
   :local:



Summary
-------

Provides an abstraction for a store which manages user accounts.











Syntax
------

.. code-block:: csharp

   public interface IUserStore<TUser> : IDisposable where TUser : class
