

ModelStateDictionary Class
==========================



.. contents:: 
   :local:



Summary
-------

Represents the state of an attempt to bind values from an HTTP Request to an action method, which includes
validation information.





Inheritance Hierarchy
---------------------


* :dn:cls:`System.Object`
