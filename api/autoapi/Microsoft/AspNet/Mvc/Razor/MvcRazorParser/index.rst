

MvcRazorParser Class
====================



.. contents:: 
   :local:



Summary
-------

A subtype of :any:`Microsoft.AspNet.Razor.Parser.RazorParser` that :any:`Microsoft.AspNet.Mvc.Razor.MvcRazorHost` uses to support inheritance of tag
helpers from <c>_ViewImports</c> files.
