

ViewContextAttribute Class
==========================



.. contents:: 
   :local:



Summary
-------

Specifies that a tag helper property should be set with the current 
:any:`Microsoft.AspNet.Mvc.Rendering.ViewContext` when creating the tag helper. The property must have a
public set method.
