

WebSocketAcceptAdapter Class
============================



.. contents:: 
   :local:



Summary
-------

This adapts the ASP.NET WebSocket Accept flow to match the OWIN WebSocket accept flow.
This enables OWIN based components to use WebSockets on ASP.NET servers.
