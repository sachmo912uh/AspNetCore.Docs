

IRouteConstraint Interface
==========================



.. contents:: 
   :local:



Summary
-------

Defines the contract that a class must implement in order to check whether a URL parameter value is valid for a constraint.











Syntax
------

.. code-block:: csharp

   public interface IRouteConstraint
